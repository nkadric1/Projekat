# Project (Projekat)
The project represents an application used by the administrative part of the company 'DeveLoop'. Administrators have insight into employees, projects and departments.
This application can be run from IntelliJ (run MainFX) and from the command line with two commands (mvn clean install and mvn clean javafx:run).
It's a Maven project. The 'App' class represents a way to manage the Command Line Interface with 4 options. It has a user defined exception - 'EmployeeException'.

Projekat predstavlja aplikaciju koju koristi administrativni dio kompanije 'DeveLoop'. Administratori imaju uvid u zaposlene, projekte i odjele.
Ova aplikacija se može pokrenuti iz IntelliJ-a (pokreni MainFX) i iz komandne linije sa dvije komande (mvn clean install i mvn clean javafx:run).
To je maven projekat. Klasa 'App' predstavlja način upravljanja interfejsom komandne linije sa 4 opcije. Ima korisnički definiran izuzetak - 'EmployeeException'.
